Should be implemented by automatically generated code of the benchmarking system for every runtime that makes use of this pallet. This trait is also implemented on (). The implemention on () is unsafe and must only be used during development. Proper weights can be generated by running the pallet_contracts benchmark suite for the runtime in question.
